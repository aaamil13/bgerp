Alt - Показване на клавишни комбинации
Alt + A - Избор на артикул
Alt + K - Подробни настройки на избрания ред на артикула
Alt + P - Плащане по бележката
Alt + C - Избор на контрагент
Alt + T - Текст към артикула
Alt + R - Преглед на бележките
Alt + F - Преглед на артикула
Alt + L - Печат на бележката
Alt + V - Отваряне на виртуална клавиатура
Alt + S - Зареждане на артикулите от сторнираната бележка
Alt + N - Оттегляне на бележка
Alt + H - Помощна информация
Alt + X - Изход терминала
Alt + D - Изтриване на селектирания ред от бележката
